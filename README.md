# FLOW EXCHANGE

Bridging On- and Off-Chain Asset Trading

# Development

1. Clone this repo

   ```bash
   git clone git@github.com:laminar-protocol/flow-exchange.git
   ```

2. Install dependencies by `yarn`

   ```bash
   yarn
   ```

3. Launch the UI
   ```bash
   yarn start
   ```
